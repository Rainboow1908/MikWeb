歡迎來至 Mik 米客之伺服！

本伺服爲創造休閒之伺服，諸模皆允（含外掛），然以不損營造、不害他人之作為前提。

## 如何入服

### 中國大陸之玩家

1. 入吾 QQ 之羣
2. 於羣內向執事請白名單
3. 待執事審而復之

### 海外之玩家

1. 填呈請之表
2. 待郵訊之復（常於四十八時辰內）
3. 審過之後，汝將得伺服之址

### 版與基巖版之說

今伺服之版爲 `26.1.2`。

- Java 版支 `1.9 ~ 26.1.2` 入，薦用 `26.1.2`
- 基巖版僅支最新之數正版

若汝以基巖版入伺服，請先竟帳之繫：

- [Geyser 帳之繫口](https://link.geysermc.org/start)

## 常見之疑

### 無法連至伺服

- **「連接被拒」** — 汝尚未得白名單，請先呈請入服
- **「未知之主機」** — 址有誤，請覈之
- **「版本不匹配」** — 請更版至 `26.1.2`

### 登入時崩潰

- 試刪除或暫停衝突之模
- 若不裝模仍崩，請更新 Java 至最新
- 仍不得解，請於羣內求助
