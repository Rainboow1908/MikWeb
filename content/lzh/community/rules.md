## 伺服之性

本伺服爲創造休閒之伺服，以營造創作與社羣往來爲心。伺服於客戶端 Mod 之用持開許之態，然諸般遊戲之行皆須以尊他作、護伺服之序爲先。

## 玩家之身與考察之制

新玩家入後，預爲**黃名**之身，居考察之期。考察期內執事將觀玩家之行。

考察之期無恆久，統據玩家之活度推計，自晉爲正員（**白名 Member**），具術不對外宣。若考察期內罹違或不宜之行，執事有權長其考察之期。

- **黃名**：新玩家預身，居考察之階，具以下限權：
  - Axiom 之能受縛
  - 不得於語中用格式之文
  - 不得用飾顱之令 `/headdb`（或 `/hdb`）
  - 不得用伺狀態詢令 `/mspt`、`/tps`
  - 不得用清除周圍落物令 `/rmitems`
  - 不得用暫白單令 `/tempwhitelist`
  - 不得用前後綴設令 `/nametag`
  - 不得用原版令 `/attribute`、`/damage`、`/effect`、`/fill`、`/item`、`/ride`、`/save-all`、`/setblock`、`/summon`
- **白名 Member**：考過後之正員身，由統據活度自晉，解上諸限

## 營造與創作之範

1. **禁毀他營**，禁毀營造必要之外之地形。反證：於野妄掘坑；削山體之半而不爲平處。
2. **營造需與周境諧**，避顯斷層或突兀之構，選址須宜。反證：於茂林中央平造大混凝方營而不爲渡；於營造密集之區緊貼他營而造格全牾之構。終之當否由執事斷。
3. **許輕紅石**，禁鉅或卡伺之紅石機，盡抑高頻之算。漏斗（Hopper）將續生效費，議免用，尤禁鉅量疊積。若有疑，請先詢執事。
4. **必於營造中當位標造者 ID 及原作**（若有參或復現他作）
5. **禁修不良影響之營**，嚴禁修含不良影響、惡營。如：無特故妄修某玩家之墓。

## Mod 與外掛之用範

伺服許玩家自用客戶端 Mod，含常義之輔類「外掛」之能，然以下之行無論以何術成之，皆**明禁**：

1. **對他玩家發攻、強移或致非願之傳**，擾其常戲
2. **毀、移、覆或竄他之營造或其所倚之地形**，含被納營造設之天然地形
3. **繞過伺服權限之制**，含行越權之令、強易他戲式等
4. **致伺服效能下損之諸行**，含頃刻鉅生實體或落物、觸大域續區塊載入、高頻重行大規操作等

> 簡之：Mod 之用，以不影他人、不毀伺服爲界。若於一法是否合規存疑，請先詢執事。

## 行爲與社羣之範

1. **持本禮**，禁於公頻發侮、攻、擾他玩家之言
2. **禁身攻與惡挑**，含對玩家之營格、美或身分之惡評
3. **禁於諸公場鉅發重或無義之文**，含語頻、告示牌、書與筆等，擾他常語
4. **禁發布與伺服無關之廣告、引流或商推之文**
5. **禁散布虛訊或惡謠**，尤涉伺服之管、他玩家之不實文

## 版權與授之聲

1. 玩家於伺服內創之營造，其著權歸創者本人有
2. 入伺服而爲創，即視允將作預用 **[CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh)（署名—非商用—同式共）** 許契發布
3. 許於非商用下復或改作，然須註原作及 Mik 米客伺服之名
4. 禁將作用於利伺、費文、地圖售或其他商用
5. 衍或改之作須續用 CC BY-NC-SA 4.0 許發布
6. 禁未許包發伺服存檔或鉅搬伺服總營之文；此條爲對 CC 契之附限，先適
7. 伺服留對作進行展、宣及社推之用權

## 官網營造收錄之範

伺服官網將不期收伺服內之優營作，以展社創之果。收錄遵以下之規：

1. **作須具一成度，且合伺服營造與創之範**，未完或有違之營不錄
2. **作須爲創者本人有**，衆人合作之作須列諸主豫者
3. **錄前須取原作明允**，作者有權拒錄或請撤已錄之作；作者請撤之時，執事當於當時內以處之。若作涉多創者而見不一，由執事調處
4. **展文須註造者 ID 及創之時**，衆人合作之作須註諸主豫者
5. **禁展含侵權文之營**，含未授復現之受版護 IP 像或營等
6. **錄文遵 CC BY-NC-SA 4.0 許契**，官網對作之展與推屬契許域內之非商用

> 執事有對上規之終解權，並有權據實情爲斷與處。
