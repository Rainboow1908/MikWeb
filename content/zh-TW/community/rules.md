## 伺服器性質

本伺服器為創造休閒向伺服器，以建築創作與社群交流為核心。伺服器對客戶端 Mod 使用持開放態度，但一切遊戲行為均須以尊重他人創作、維護伺服器秩序為前提。

## 玩家身份與考察制度

新玩家加入後預設為**黃名**身份，處於考察期。考察期內管理員會對玩家的行為進行觀察。

考察期無固定時長，系統將根據玩家的活躍度進行計算並自動晉升為正式成員（**白名 Member**），具體演算法不對外公開。若在考察期內出現違規或不當行為，管理員有權延長考察期限。

- **黃名**：新玩家預設身份，處於考察階段，具有以下權限限制：
  - Axiom 功能受限
  - 無法在聊天中使用格式化文字
  - 無法使用裝飾頭顱獲取指令 `/headdb`（或 `/hdb`）
  - 無法使用伺服器狀態查詢指令 `/mspt`、`/tps`
  - 無法使用清除周圍掉落物指令 `/rmitems`
  - 無法使用臨時白名單指令 `/tempwhitelist`
  - 無法使用前後綴設定指令 `/nametag`
  - 無法使用原版指令 `/attribute`、`/damage`、`/effect`、`/fill`、`/item`、`/ride`、`/save-all`、`/setblock`、`/summon`
- **白名 Member**：考察通過後的正式成員身份，由系統根據活躍度自動晉升，解除上述限制

## 建築與創作規範

1. **禁止破壞他人建築**，禁止破壞建築必要之外的地形。反例：在野外隨意挖坑；將山體削去一半後不做平滑處理。
2. **建築需與周圍環境協調**，避免明顯斷層或突兀結構，選址需合理。反例：在植被茂密的森林中央平地建造大型混凝土方塊建築而不做任何過渡；在已有建築密集的區域緊貼他人建築建造風格完全衝突的結構。最終是否合適由管理員判斷。
3. **允許輕量紅石**，禁止大規模或卡服紅石裝置，儘量降低高頻運算。漏斗（Hopper）會持續產生效能開銷，建議避免使用，尤其禁止大量堆疊。如有疑問請提前詢問管理員。
4. **必須在建築中合理位置標註建築者 ID 及原作者**（如有參考或復現他人作品）
5. **禁止修建不良影響建築**，嚴禁修建含不良影響、惡意建築。例如：無特殊情況故意修建某位玩家的墓地。

## Mod 與外掛使用規範

伺服器允許玩家自由使用客戶端 Mod，包括通常意義上的輔助類「外掛」功能，但以下行為無論透過何種方式實現均**明確禁止**：

1. **對其他玩家發動攻擊、強制位移或造成非自願傳送**，干擾其正常遊戲
2. **破壞、移動、覆蓋或修改他人的建築或其所依託的地形**，包括被納入建築設計的自然地形
3. **繞過伺服器權限限制**，包括執行越權指令、強制改變他人遊戲模式等
4. **導致伺服器效能下降的一切行為**，包括短時間內大量生成實體或掉落物、觸發大範圍持續區塊載入、高頻重複執行大規模操作等

> 簡而言之：Mod 的使用以不影響他人、不破壞伺服器為界限。如對某一用法是否合規存有疑問，請提前詢問管理員。

## 行為與社群規範

1. **保持基本禮貌**，禁止在公共頻道發表侮辱、攻擊、騷擾其他玩家的言論
2. **禁止人身攻擊與惡意挑釁**，包括針對玩家的建築風格、審美或個人身份的惡意評價
3. **禁止在任何公共場合大量傳送重複或無意義內容**，包括聊天頻道、告示牌、書與筆等，干擾他人正常交流
4. **禁止發佈與伺服器無關的廣告、引流或商業推廣內容**
5. **禁止散佈虛假資訊或惡意謠言**，尤其是涉及伺服器管理、其他玩家的不實內容

## 版權與授權聲明

1. 玩家在伺服器內創作的建築作品，其著作權歸創作者本人所有
2. 進入伺服器並進行創作，即視為同意將作品預設採用 **[CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh)（署名—非商業性使用—相同方式共享）** 許可協議發佈
3. 允許在非商業用途下複製或改編作品，但必須註明原作者及 Mik 米客伺服器名稱
4. 禁止將作品用於盈利伺服器、付費內容、地圖售賣或其他商業用途
5. 衍生或改編作品必須繼續採用 CC BY-NC-SA 4.0 許可發佈
6. 禁止未經許可打包發佈伺服器存檔或大規模搬運伺服器整體建築內容；此條為對 CC 協議的附加限制，優先適用
7. 伺服器保留對作品進行展示、宣傳及社群推廣的使用權

## 官網建築收錄規範

伺服器官網將不定期收錄伺服器內的優質建築作品，以展示社群創作成果。收錄遵循以下規則：

1. **作品須具備一定完成度，且符合伺服器建築與創作規範**，未完成或存在違規的建築不予收錄
2. **作品須為創作者本人所有**，多人合作作品須列明所有主要參與者
3. **收錄前須取得原作者明確同意**，作者有權拒絕收錄或申請撤除已收錄的作品；作者申請撤除時，管理員應在合理時間內予以處理。若作品涉及多位創作者且意見不一，由管理員協調處理
4. **展示內容須註明建築者 ID 及創作時間**，多人合作作品須註明所有主要參與者
5. **禁止展示含有侵權內容的建築**，包括未經授權復現的受版權保護的 IP 形象或建築等
6. **收錄內容遵循 CC BY-NC-SA 4.0 許可協議**，官網對作品的展示與推廣屬於協議允許範圍內的非商業性使用

> 管理員擁有對上述規則的最終解釋權，並有權根據實際情況作出判斷與處置。
