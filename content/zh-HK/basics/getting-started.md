歡迎來到 Mik 米客伺服器！

本伺服器為創造休閒向伺服器，允許任意 Mod（包含外掛），但前提是不破壞建築與他人創作。

## 如何加入伺服器

### 中國大陸玩家

1. 加入我們的 QQ 羣
2. 在羣聊內向管理員申請白名單
3. 等待管理員審核並回覆

### 海外玩家

1. 填寫申請表單
2. 等待接收郵件通知（通常在 48 小時內）
3. 審核通過後，你會收到伺服器地址

### 版本與基岩版說明

當前伺服器版本為 `26.1.2`。

- Java 版支援 `1.9 ~ 26.1.2` 進入，推薦使用 `26.1.2`
- 基岩版僅支援最新幾個正式版

如你使用基岩版進入伺服器，請先完成帳號綁定：

- [Geyser 帳號綁定入口](https://link.geysermc.org/start)

## 推薦模組

為了獲得更佳的遊戲體驗，我們推薦以下兩款優化整合包，按需選擇：

| | [Fabulously Optimized](https://modrinth.com/modpack/fabulously-optimized) | [Rapid Optimization](https://modrinth.com/modpack/rapid_optimization) |
|---|---|---|
| **效能** | 效能不錯 | 效能更高 |
| **功能** | 更多常用功能 | 精簡純淨 |

在此基礎上，再安裝以下模組以獲得完整體驗：

1. **[PlasmoVoice](https://modrinth.com/plugin/plasmo-voice)** - 語音聊天與聽歌
2. **Xaero 地圖系列** - 伺服器已專門適配
   - [Xaero's World Map](https://modrinth.com/mod/xaeros-world-map)
   - [Xaero's Minimap](https://modrinth.com/mod/xaeros-minimap)
   - [XaeroPlus](https://modrinth.com/mod/xaeroplus)
3. **[ImageFrame Client](https://modrinth.com/mod/imageframeclient)** - 地圖畫彩色增強
4. **[Axiom](https://modrinth.com/mod/axiom)** - 強大的建築輔助模組
5. **[Vivecraft](https://modrinth.com/mod/vivecraft)** - VR 支援

剩下是一些伺服器無關的推薦模組：

- **[Hey Wiki](https://modrinth.com/mod/hey-wiki)** - 便捷的 Wiki 查詢工具
- **[voxy](https://modrinth.com/mod/voxy)** - 效能極高的遠距離區塊渲染模組，獲得更佳的風景

## 首次進入伺服器

當你第一次進入伺服器時：

- 你會出生在主城區域
- 使用 `/spawn` 可以隨時返回主城
- 使用 `/sethome` 設定你的家
- 使用 `/home` 回到你的家

## 開始創作

使用你的小地圖模組，到處跑圖，瞭解伺服器建築概況，尋找你喜歡的地方和建築羣。

找到一個你喜歡的地方，開始你的建築創作：

1. 選擇合適的建造地點
2. 確保建築與周圍環境協調
3. 使用 `/sethome [名稱]` 設定家的位置
4. 開始你的創造之旅！

> **邊界提示**：主世界邊界直徑為 `6144` 格，請在邊界範圍內規劃你的建築位置。
>
> **提示**：伺服器支援 Axiom 和創世神（WorldEdit），可以大大提升建築效率！
